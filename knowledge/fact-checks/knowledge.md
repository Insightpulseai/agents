- Code was rapidly copied, reposted, forked, and mirrored on GitHub
- Key repo: `instructkr/claw-code` — original uploader pivoted to a clean-room Python rewrite citing legal liability concerns, now Rust-first
- Repository grew to 106k+ stars and 95k+ forks (The Verge reported 50k+ forks shortly after spread)
- Derivative works include Python and Rust rewrites using AI coding tools (Codex)
